# Simple Employee Data Entry Form in Excel

## Overview

This is a simple Employee Data Entry Form created using Microsoft Excel. The form allows users to enter employee details and store them in a structured database sheet.

The project was created to practice Excel form design, data organization, and record management.

## Features

* Employee information entry form
* Stores records in a database sheet
* Organized tabular data storage
* Salary details and net pay fields
* Simple and easy-to-use interface

## Data Captured

* Employee Name
* Employee Code
* Email ID
* Date of Birth
* Department
* Location
* Basic Salary
* HRA
* Medical
* Conveyance
* Deductions
* Net Pay

## Tools Used

* Microsoft Excel
* Excel Functions
* Form Controls
